## How to Play

1. Open `index.html` in any web browser
2. Click "Start Game"
3. Use arrow keys or touch buttons to control the snake
4. Eat the red food to grow longer
5. Don't hit the walls or yourself!

## Controls

- **Keyboard**: Arrow keys or WASD
- **Touch**: On-screen buttons or swipe on the canvas
- **Mobile**: Swipe in any direction to move

## Running on Mobile

### Option 1: Direct Browser
1. Copy all files to your phone
2. Open `index.html` in Chrome or any browser
3. Add to home screen for app-like experience

### Option 2: Local Server
1. Run a local server: `python -m http.server 8000`
2. Open on phone: `http://YOUR-IP:8000`

### Option 3: Deploy Online
Deploy to GitHub Pages, Netlify, or Vercel for easy mobile access!

## Converting to Android APK

You can convert this to a real Android app using:
- **Cordova**: `cordova create`, add files, build APK
- **Capacitor**: Modern alternative to Cordova
